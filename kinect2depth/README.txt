Introduction
============

A simple tool to record depth streams from the Kinect camera.

Requires
- Visual Studio Express 2010 C#
- XNA 4.0
- Kinect SDK (Toolkit and Runtime are optional)
- XBox gamepad


Usage
=====

1. Start the app.
2. Press 'A' on the gamepad once to start and once stop recording.
3. A file dialog pops up. Choose where you want to save the recording to.
4. Quit the app with 'Back' or 'ESC'.

This tool saves '.depth' files as described in poly2depth/readme.txt.
